# Dosya depolama düzeni

Bu klasör yalnızca yol sözleşmesini belgeler. Canlı dosyalar Git'e veya yerel uygulama klasörüne değil, `FILES` nesne depolama alanına yazılır.

```text
public/course-resources/{courseId}/{academicYear}/{versionId}/
public/syllabus-pdfs/{courseId}/{academicYear}/{versionId}/
private/draft-attachments/{courseId}/{versionId}/
private/approval-documents/{approvalId}/
```
