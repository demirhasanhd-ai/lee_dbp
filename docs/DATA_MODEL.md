# Başlangıç veri modeli

Temel varlıklar: `institutes`, `departments`, `programs`, `courses`, `course_versions`, `learning_outcomes`, `weekly_topics`, `assessment_items`, `workload_items`, `program_outcomes`, `outcome_matrix`, `approval_requests`, `revision_comments`, `audit_logs`, `files`, `users` ve `roles`.

`courses` dersin değişmeyen kimliğini, `course_versions` ise akademik yıl ve sürüme bağlı Bologna içeriğini tutar. e-Enstitü kaynaklı kayıtlarda yerel kimliğe ek olarak `external_id` bulunur; ders kodu benzersiz kimlik olarak kullanılmaz.
