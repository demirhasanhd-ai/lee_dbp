# bologna.universite.edu.tr

Kurumlar üstü veya üniversite geneli Bologna portalı için ayrılan alan adı katmanıdır.

`dbp.osmaniye.edu.tr` uygulamasının katalog bağlantısı, kurum kimliği ve API uç noktası bu katmanda tanımlanabilir. Bu klasör çalışma zamanında dosya yükleme alanı olarak kullanılmaz.
